# Demoday
여행 정산 서비스 두더치(DoDutch)

## 프로젝트 소개
친구들끼리 또는 단체로 여행을 갈 때, 정산 과정에서 어려움을 겪는 경우가 많습니다. 
<br>
이러한 불편함을 해결하기 위해 두더치(DoDutch) 앱 서비스를 기획했습니다. 이 서비스는 여행에 참여하는 사용자가 여러 결제 내역을 기록하면, 이를 바탕으로 자동으로 정산을 수행합니다.
